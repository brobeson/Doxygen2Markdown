# `dox_md` Command Reference

```bash
usage: python3 dox_md.py [-h] [--log-level {debug,info,warning,error,critical}] input output

Convert Doxygen XML output to Markdown files.

positional arguments:
  input                 The path to the directory containing Doxygen XML.
  output                Path to the directory to write Markdown files.

options:
  -h, --help            show this help message and exit
  --log-level {debug,info,warning,error,critical}
                        Log messages at this severity and higher.
```
